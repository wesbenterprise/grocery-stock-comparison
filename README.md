# Grocery Stock Comparison — Q4 2025

Static dashboard comparing Publix, Walmart, and Kroger based on latest available quarter data.

## Notes
- Publix is private: some values are estimates.
- Walmart uses Q4 FY2026 (ended Jan 2026).
- Kroger Q4 FY2025 not yet reported; dashboard uses Q3 FY2025 (ended Nov 2025).

## Run locally
Open `index.html` directly, or serve folder with any static server.

## Deploy
Optimized for Vercel static deployment.
